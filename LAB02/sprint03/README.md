# Laboratório de Experimentação de Software
---
## Processo de desenvolvimento

Na terceira sprint, foi realizada a consolidação final da análise, com refinamento dos resultados obtidos, validação das hipóteses e organização do relatório final. Também foram aplicadas melhorias na interpretação dos dados e ajustes metodológicos para garantir maior confiabilidade nas conclusões.

- `dados_tratados.csv`: versão final dos dados após limpeza, normalização e preparação para análise.
- `analise_final.py`: script utilizado para refinar os cálculos, validar métricas e gerar os resultados finais.
- `graficos_finais/`: pasta com os gráficos definitivos utilizados no relatório (correlações e distribuições das métricas).
- `relatorio_final.pdf`: documento final contendo introdução, metodologia, resultados e discussão completa do estudo.
- `apresentacao.pdf`: slides preparados para apresentação dos resultados em sala.
