# Laboratório de Experimentação de Software
---
## Processo de desenvolvimento

Na segunda sprint, foi realizada a coleta completa das métricas para todos os repositórios selecionados, seguida da consolidação dos dados e início da análise. Também foram definidas hipóteses e geradas visualizações para apoiar a interpretação dos resultados.

- `repositorios_completo.csv`: contém os dados consolidados dos 1000 repositórios Java, incluindo métricas de processo e qualidade.
- `analise_dados.py`: script responsável por processar os dados, calcular estatísticas (média, mediana e desvio padrão) e preparar os dados para análise.
- `graficos/`: pasta com os gráficos gerados (correlações entre métricas como popularidade, tamanho, atividade e qualidade).
- `hipoteses.txt`: documento contendo as hipóteses definidas com base nas questões de pesquisa.
- `relatorio_final.md`: documento com a análise dos resultados, discussão das hipóteses e conclusões do estudo.
